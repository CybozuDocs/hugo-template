<Kintone />システム管理の操作をしたときに出力されるログについて説明します。

<Heading id="log_system_administration_10,class=&quot;no-trans&quot;">System administration</Heading>

モジュールが「System administration」である監査ログは、下記のとおりです。

<Heading level={3} id="log_system_administration_1010,class=&quot;no-trans&quot;">Admit creation space</Heading>

アクションが「Admit creation space」である監査ログについて説明します。

<Heading level={4} id="log_system_administration_101010">実行された操作</Heading>

ゲストスペースの作成権限の設定を変更した。

<Heading level={4} id="log_system_administration_101020">レベル</Heading>

重要

<Heading level={4} id="log_system_administration_101030">「補足」に出力される内容</Heading>

次の項目が出力されます。

* ゲストスペースの作成を許可されたユーザー、組織、またはグループ
* ゲストスペースの作成を制限されたユーザー、組織、またはグループ

対象は、ログイン名、組織コード、またはグループコードで出力されます。

<Heading level={4} id="log_system_administration_101040">出力されるログの例</Heading>

```
granted users: [*], revoked users: [*]

```

<Heading level={3} id="log_system_administration_1012,class=&quot;no-trans&quot;">Guest user two-step verification</Heading>

アクションが「Guest user two-step verification」である監査ログについて説明します。

<Heading level={4} id="log_system_administration_101210">実行された操作</Heading>

<Heading level={4} id="log_system_administration_101220">レベル</Heading>

重要

<Heading level={4} id="log_system_administration_101230">「補足」に出力される内容</Heading>

次の項目が出力されます。

* 二段階認証を有効にした場合：enabled
* 二段階認証を無効にした場合：disabled

<Heading level={4} id="log_system_administration_101240">出力されるログの例</Heading>

```

```
