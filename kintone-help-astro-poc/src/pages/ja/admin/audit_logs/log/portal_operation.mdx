ポータルの操作をしたときに出力されるログについて説明します。

<Heading id="log_portal_operation_10,class=&quot;no-trans&quot;">Portal operation</Heading>

モジュールが「Portal operation」である監査ログは、下記のとおりです。

<Heading level={3} id="log_portal_operation_1010,class=&quot;no-trans&quot;">Portal announcement file download</Heading>

アクションが「Portal announcement file download」である監査ログについて説明します。

<Heading level={4} id="log_portal_operation_101020">実行された操作</Heading>

お知らせ掲示板に添付されたファイルについて、次の操作をした。

* ファイルをダウンロードした
* 画像のスライドショーを表示した

<Heading level={4} id="log_portal_operation_101030">レベル</Heading>

情報

<Heading level={4} id="log_portal_operation_101040">「補足」に出力される内容</Heading>

ファイル名が出力されます。

<Heading level={4} id="log_portal_operation_101050">出力されるログの例</Heading>

```
filename: *

```

<Heading id="log_portal_operation_20">プロパティ</Heading>

監査ログの「補足」欄に出力されるプロパティは次のとおりです。

<Heading level={3} id="log_portal_operation_2010,class=&quot;no-trans&quot;">filename</Heading>

ファイル名です。
