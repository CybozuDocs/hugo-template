<Heading id="log_guest_10,class=&quot;no-trans&quot;">Guest management</Heading>

モジュールが「Guest management」である監査ログは、下記のとおりです。

<Heading level={3} id="log_guest_1010,class=&quot;no-trans&quot;">Invite guest</Heading>

アクションが「Invite guest」である監査ログについて説明します。

<Heading level={4} id="log_guest_101010">実行された操作</Heading>

ゲストユーザーに招待メールを送信した。

<Heading level={4} id="log_guest_101020">レベル</Heading>

重要

<Heading level={4} id="log_guest_101030">「補足」に出力される内容</Heading>

次の項目が出力されます。

* スペースID
* スペース名
* メールアドレス

<Heading level={4} id="log_guest_101040">出力されるログの例</Heading>

```
space id: *, space name: *, Email: [*]

```

メールアドレスはカンマ区切りで表示されます。

<Heading id="log_guest_40">ゲストのアカウント登録時に記録されるログ</Heading>

ゲストは、次のいずれかの方法でアカウントを登録して、招待されたゲストスペースに参加できます。

* kintoneアカウントを使用して登録する（招待したゲストが別のドメインでkintoneを利用している場合）
* 新規ゲストアカウントを登録する

ゲストがkintoneアカウントを使用して登録した場合、アクションが「Guest integrate account」のログが記録されます。  
ゲストが新規ゲストアカウントを登録した場合、アクションが「Guest sign up」のログが記録されます。  
ゲストがkintoneアカウントを使用して登録した場合、そのゲストの利用料金は無料になります。
