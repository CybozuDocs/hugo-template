<Heading id="log__index10">ゲストユーザーによる操作</Heading>

ゲストユーザーが操作をした場合、「ユーザー」に「[nobody]」が出力されます。  
「補足」の先頭には、ゲストユーザーのログイン名が出力されます。

```
login name: *
```
