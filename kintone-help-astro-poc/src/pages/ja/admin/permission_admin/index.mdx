権限や設定項目の例と、それらの設定画面は以下のとおりです。詳細については、各リンク先を参照してください。

* <Kintone />全体に関わる権限
  * 権限の例：アプリやスペースの作成権限、アプリグループの管理権限、システムの管理権限など
  * 設定画面：<Wovn>［<Kintone />システム管理］</Wovn>画面で設定できます。  
  [<Kintone />システム管理で設定できるアクセス権](/k/ja/id/04062.html)  
  [アプリグループの管理](/k/ja/id/04023.html)
* アプリごとに設定する権限
  * 権限の例：レコード追加／編集の権限、フィールドごとの閲覧権限など
  * 設定画面：<Wovn>［アプリの設定］</Wovn>画面で設定できます。  
  [アクセス権の設定](/k/ja/id/040588.html)
* スペースごとに設定する権限
  * 権限の例：スペース内アプリの作成権限、スペースの管理権限など
  * 設定画面：<Wovn>［スペースの管理］</Wovn>ダイアログで設定できます。  
  [スペースの設定を変更する](/k/ja/id/040696.html)
* <Kintone />を安全に運用するための設定
  * 設定項目の例：組織／ユーザーの設定、組織間のアクセス権など
  * 設定画面：<SlashUi />で設定できます。  
  [<SlashUi />での権限設定](/k/ja/id/04059.html)  
  [<SlashHelp />](/general/ja/id/0202.html)
