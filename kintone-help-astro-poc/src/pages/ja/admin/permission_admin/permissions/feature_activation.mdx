<Wovn>［利用する機能の選択］</Wovn>画面では、一部の機能を無効化および再有効化できます。

<Heading level={3} id="admin_feature_activation_3010">スペースに所属しないアプリの作成を許可する</Heading>

この機能は、スペースに所属しないアプリの作成を許可するかどうかを設定できる機能です。

アプリは、ポータル（トップページ）またはスペース内のいずれかに配置されます。スペースに所属しないアプリとは、ポータルに配置されるアプリのことです。

* **チェックを入れている場合：**  
  スペースに所属するアプリ（スペース内アプリ）／所属しないアプリのどちらも作成できます。
* **チェックを外している場合：**  
  スペースに所属するアプリ（スペース内アプリ）しか作成できなくなります。スペースに所属しないアプリは作成できません。なお、チェックを外した場合でも、作成済みのアプリには影響ありません。

[スペース内アプリとは](/k/ja/id/040709.html)

<Heading level={3} id="admin_feature_activation_3020">参考：&lt;Wovn&gt;［スペースに所属しないアプリの作成を許可する］&lt;/Wovn&gt;を無効にした場合のアプリグループ</Heading>

アプリは、必ずいずれかのアプリグループが設定されています。アプリ作成時は<Wovn>［デフォルトのアプリグループ］</Wovn>で設定しているアプリグループが選択されますが、ほかのアプリグループに変更することも可能です。ただし、スペースに所属するアプリ（スペース内アプリ）は必ず<Wovn>［Public］</Wovn>というアプリグループとなり、変更できません。

<Wovn>［スペースに所属しないアプリの作成を許可する］</Wovn>のチェックを外すと、スペースに所属するアプリ（スペース内アプリ）しか作成できなくなります。このため、アプリを作成するとすべてPublicアプリグループとなり、変更できません。つまり、<Wovn>［スペースに所属しないアプリの作成を許可する］</Wovn>のチェックを外すと、Publicグループ以外（Privateグループを含む）のアプリ作成は許可しないという状態になります。

Publicグループ以外のアプリグループを活用している場合は、上記を考慮して設定してください。
