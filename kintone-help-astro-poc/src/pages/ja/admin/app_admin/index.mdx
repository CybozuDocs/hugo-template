<Kintone />のシステム管理者は、<Wovn>［<Kintone />システム管理］</Wovn>の<Wovn>［アプリ］</Wovn>の各項目から、アプリやアプリテンプレートの管理ができます。

* <Wovn>［アプリ管理］</Wovn>画面では、ユーザーが作成したアプリの利用状況を確認できます。アプリの一覧から、<Wovn>［アプリの設定］</Wovn>画面を開くことや、アプリを削除することも可能です。  
  また、削除してしまったアプリの復旧方法についても案内しています。  
  [<Wovn>［アプリ管理］</Wovn>画面でできること](/k/ja/id/04015.html)  
  [削除したアプリを復旧する](/k/ja/id/04016.html)
* <Wovn>［アプリテンプレート］</Wovn>画面では、アプリテンプレート（ひな型）の作成や、ファイルへの書き出し、ファイルからの読み込みができます。  
  また、アプリテンプレートに含まれない設定についても案内しています。テンプレート作成前に、あらかじめご確認ください。  
  [アプリテンプレートの管理](/k/ja/id/04017.html)  
  [アプリテンプレートに含まれない設定](/k/ja/id/04021.html)
