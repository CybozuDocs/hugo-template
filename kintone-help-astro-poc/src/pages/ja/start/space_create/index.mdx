スペースとは、必要なやり取りや情報を集約できる「チームの場所」のようなものです。チームにとって必要な情報や連絡事項などを集約できます。  

ここでは、<Kintone />を使い始めたばかりの方向けに、スペースの基本操作をイメージできるような、簡単なスペースの作成方法、およびスペース上でチームメンバーとやり取りする方法を紹介します。
