<Kintone />を複数人で利用するには、利用するユーザーを追加する必要があります。また、部署などを「組織」として追加し、複数のユーザーを管理することも可能です。

<Kintone />にユーザーや組織を追加するには、次の方法があります。

* **招待機能を利用してメンバーを追加する：**  
  はじめてログインしたときや無料お試し期間中は、招待機能を利用してほかのメンバーにメールを送ることで、<Kintone />にユーザーが追加されます。詳細は、次のページを参照してください。  
  [チームメンバーを招待する](/k/ja/id/040139.html)
* **ユーザーと組織を登録する：**  
  お試し期間中であるかどうかにかかわらず、ユーザーや組織を［<SlashUi />］画面から登録できます。詳細は、次のページを参照してください。  
  [組織を追加する](/k/ja/id/040137.html)  
  [ユーザーを追加する](/k/ja/id/040138.html)
