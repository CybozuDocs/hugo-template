<Kintone />では、さまざまな業務に対応するアプリを作成することが可能です。アプリを作成するには、ドラッグアンドドロップで必要な項目を選んで作成する方法や、今使っているExcelファイルからアプリを作成する方法などがあります。

ここでは、<Kintone />を使い始めたばかりの方向けに、アプリの基本操作をイメージできるような簡単なアプリの作成や設定変更の方法、アプリにデータを登録する手順などを紹介します。実際に操作しながら、<Kintone />のアプリの構成や特徴を体験してみましょう。
