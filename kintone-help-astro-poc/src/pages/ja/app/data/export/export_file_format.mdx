レコードのデータをファイルに書き出すと、レコードやフィールドの値がどのように出力されるかについて補足します。

<Heading id="export_export_file_format_10">レコード</Heading>
1レコード1行でファイルに出力されます。  
ただし、テーブルが含まれるレコードは、1レコードの内容が複数行で出力されます。  
ファイルの先頭列に出力された「レコードの開始行」の「\*」から次の「\*」の前までの行が、1つのレコードのデータです。  
テーブル以外の値は、1レコード内の行に繰り返しコピーされた状態で出力されます。  
<Img src="/k/kintone/img-ja/export_file_format_01.png" alt="スクリーンショット：出力されたファイルで1レコードの範囲を示している" />

<Heading id="export_export_file_format_20">フィールド</Heading>
<Heading level={3} id="export_export_file_format_2010">リッチエディター</Heading>
リッチエディターフィールドに入力されている一部の文字は、文字参照で出力されます。  
文字参照で出力される文字の例は、次のとおりです。

* 全角の記号の一部（＆￥（）＊＋－／：；＜＝＞？＠～｜ など）
* 半角の記号の一部
* 全角の英数字
* 半角カタカナ など

上記の記号や文字以外にも文字参照で出力される場合があります。  
文字を閲覧・加工しやすい形式（テキスト形式）で出力したい場合は、出力オプションの「リッチエディターをテキスト形式で出力する」にチェックを入れて、ファイルに書き出してください。

<Heading level={3} id="export_export_file_format_2020">数値</Heading>
桁区切りのカンマや単位記号は、ファイルに出力されません。

<Heading level={3} id="export_export_file_format_2030">計算</Heading>
計算結果の値がファイルに出力されます。単位記号は出力されません。

<Heading level={3} id="export_export_file_format_2040">チェックボックス</Heading>
チェックボックスフィールドの値は、選択肢ごとに列が分かれます。選択された選択肢には「1」と出力されます。  
<Img src="/k/kintone/img-ja/export_file_format_02.png" alt="スクリーンショット：選択された項目の値が、選択肢ごとの列に分かれてファイルに出力されている" />

<Heading level={3} id="export_export_file_format_2050">複数選択</Heading>
複数選択フィールドの値は、選択肢ごとに列が分かれます。選択された選択肢には「1」と出力されます。  
<Img src="/k/kintone/img-ja/export_file_format_02.png" alt="スクリーンショット：選択された項目の値が、選択肢ごとの列に分かれてファイルに出力されている" />

<Heading level={3} id="export_export_file_format_20110">日付</Heading>
日付の形式は、ファイルを書き出すユーザーの表示言語の設定や、Webブラウザーの言語設定によって異なります。  
詳細は、次の項目を参照してください。

日時に含まれる日付の形式は、ファイルを書き出すユーザーの表示言語の設定や、Webブラウザーの言語設定によって異なります。

<Heading level={3} id="export_export_file_format_2060">ユーザー選択</Heading>
* ゲストユーザーのログイン名には、ログイン名の先頭に「guest/」が付与されます。
* 出力オプションの「ユーザー、組織、グループを表示名で出力する」にチェックを入れた場合、ユーザー選択フィールドの値をユーザーの表示名で出力できます。  
  ただし、組織間のアクセスが禁止されていると、閲覧権限のないユーザーの表示名には、ログイン名の後に「（表示名を取得できませんでした）」が付与された値が出力されます。
* 削除されたユーザーを出力する場合、ログイン名の後に、そのユーザーが削除された時刻がUTC（協定世界時）形式で付与されます。  
  ただし、出力オプションの「ユーザー、組織、グループを表示名で出力する」にチェックを入れた場合、表示名だけが出力されます。

<Heading level={3} id="export_export_file_format_2070">組織選択</Heading>
出力オプションの「ユーザー、組織、グループを表示名で出力する」にチェックを入れた場合、組織選択フィールドの値を組織の表示名で出力できます。  
ただし、組織間のアクセスが禁止されていると、閲覧権限のない組織の表示名には、組織コードの後に「（表示名を取得できませんでした）」が付与された値が出力されます。

<Heading level={3} id="export_export_file_format_2080">グループ選択</Heading>
出力オプションの「ユーザー、組織、グループを表示名で出力する」にチェックを入れた場合、グループ選択フィールドの値をグループの表示名で出力できます。  
ただし、組織間のアクセスが禁止されていると、閲覧権限のないグループの表示名には、グループコードの後に「（表示名を取得できませんでした）」が付与された値が出力されます。

<Heading id="export_export_file_format_40">出力される日付の形式</Heading>
月や日が1桁の場合は、数字がゼロ埋めされます。  
また、日付を含むフィールドにおける日付の形式は、ファイルを書き出すユーザーの表示言語の設定や、Webブラウザーの言語設定によって異なります。  
詳細は、次のとおりです。

<Heading level={3} id="export_export_file_format_4020">年/月/日</Heading>
日付が「年/月/日」の形式で出力される条件は、次のどちらかです。

* アカウント設定の「言語」が「英語」以外（例：日本語）
* アカウント設定の「言語」が「Webブラウザーの設定に従う」、かつ、Webブラウザーの言語設定が、次のどちらにも該当しない場合
  * 英語（アメリカ合衆国）
  * 英語

<Heading level={3} id="export_export_file_format_4010">月/日/年</Heading>
日付が「月/日/年」の形式で出力される条件は、次のどちらかです。

* アカウント設定の「言語」が「英語」
* アカウント設定の「言語」が「Webブラウザーの設定に従う」、かつ、Webブラウザーの言語設定が、次のどちらかに該当する場合
  * 英語（アメリカ合衆国）
  * 英語

<Heading id="export_export_file_format_30">困ったときは</Heading>
<Heading level={3} id="export_export_file_format_3010">書き出したいレコードのデータがファイルにありません</Heading>
レコードの絞り込み条件に該当レコードが含まれていない可能性があります。  
ファイルに書き出されるレコードは、レコード一覧画面に表示されているレコードです。  
レコードの絞り込み条件を再設定するなどして、レコード一覧画面に該当レコードが表示されていることを確認してから、ファイルに書き出してください。

<Heading level={3} id="export_export_file_format_3020">ファイルに書き出されたデータが文字化けしています</Heading>
文字コードの指定が誤っている可能性があります。詳細は、次のページを参照してください。  
[ファイル書き出し時の文字コードの選びかたを知りたい](/k/ja/id/040169.html)
