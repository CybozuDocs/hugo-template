演算子やSUM関数を使うことで、テーブル内のフィールドを計算できます。このページでは、次の2つの計算方法を説明します。

* [テーブル内で行ごとに計算する](#autocalc_table_autocalc_10)
* [テーブル内の数値や時間を合計する](#autocalc_table_autocalc_20)

<Note>


テーブルの中に指定した文字列と完全一致するものがあるかどうかは、CONTAINS関数を使って判定できます。  

詳細は次のページを参照してください。  

[[CONTAINS関数]フィールドの中に、指定した文字列と完全一致するものがあるか判定する](/k/ja/id/040506.html)


</Note>

<Heading id="autocalc_table_autocalc_10">テーブル内で行ごとに計算する</Heading>
テーブル内で行ごとに自動で計算できます。

<Heading level={3} id="autocalc_table_autocalc_1010">テーブル内で行ごとに数値を計算する例</Heading>
次の例は、テーブル内の「数量」フィールドと「単価」フィールドを自動で乗算して、金額を計算する計算式です。  
テーブル内に計算フィールドまたは文字列（1行）フィールドを配置して、次の計算式を指定します。

```plaintext {copy=true}
数量*単価
```

<Img src="/k/kintone/img-ja/table_autocalc_18.png" alt="スクリーンショット：テーブル内の「数量」と「単価」フィールドを使って金額を自動で計算している" />

この計算式を設定した「金額」フィールドに、「数量」フィールドと「単価」フィールドを乗算した金額が表示されます。

<Heading level={3} id="autocalc_table_autocalc_1020">テーブル内で行ごとに時間を計算する例</Heading>
次の例は、テーブル内の「開始」フィールドと「終了」フィールドを使って、問い合わせ対応にかかった時間を自動で表示する計算式です。  
テーブル内に計算フィールドを配置し、次の計算式を指定して、計算結果の表示形式を「時間」に設定します。

```plaintext {copy=true}
終了-開始
```

<Img src="/k/kintone/img-ja/table_autocalc_12.png" alt="スクリーンショット：テーブル内の「開始」と「終了」フィールドを使って対応時間を自動で計算している" />

この計算式を設定した「対応時間」フィールドに、「終了」フィールドから「開始」フィールドを減算した時間が表示されます。

<Note>


計算フィールドまたは文字列（1行）フィールドでは、時間だけでなく、日時や日付の計算もできます。  

詳細は、次のページを確認してください。  

[日付、時刻、日時の計算](/k/ja/id/040502.html)


</Note>

<Heading id="autocalc_table_autocalc_20">テーブル内の数値や時間を合計する</Heading>
テーブル内にあるフィールドの値を合計できます。

<Heading level={3} id="autocalc_table_autocalc_2010">テーブル内の数値を合計する例</Heading>
次の例は、テーブル内の「小計」フィールドを自動で合計する計算式です。  
テーブルの外に計算フィールドまたは文字列（1行）フィールドを配置して、次の計算式を指定します。

```plaintext {copy=true}
SUM(小計)
```

<Img src="/k/kintone/img-ja/table_autocalc_11.png" alt="スクリーンショット：テーブル内の「小計」フィールドを自動で合計している" />

この計算式を設定した「合計金額」フィールドに、「小計」フィールドの合計が表示されます。

<Note>


計算対象に未入力の項目があっても合計されます。また、合計金額にもその結果が反映されます。

<Img src="/k/kintone/img-ja/table_autocalc_13.png" alt="スクリーンショット：計算対象に未入力の項目があっても計算結果が表示されている" />


</Note>

<Heading level={3} id="autocalc_table_autocalc_2020">テーブル内の時間を合計する例</Heading>
次の例は、テーブル内の「対応時間」フィールドを自動で合計する計算式です。  
テーブルの外に計算フィールドを配置し、次の計算式を指定して、計算結果の表示形式を「時間」に設定します。

```plaintext {copy=true}
SUM(対応時間)
```

<Img src="/k/kintone/img-ja/table_autocalc_14.png" alt="スクリーンショット：テーブル内の「対応時間」フィールドを自動で合計している" />

この計算式を設定した「対応時間合計」フィールドに、「対応時間」フィールドの合計が表示されます。

<Note>


計算フィールドまたは文字列（1行）フィールドでは、時間だけでなく、日時や日付の計算もできます。  

詳細は、次のページを確認してください。  

[日付、時刻、日時の計算](/k/ja/id/040502.html)


</Note>
