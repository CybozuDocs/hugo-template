YEN関数を設定すると、計算結果を、指定した桁数で四捨五入し、3桁ごとの桁区切りの「&yen;（円）」形式で表示できます。  
文字列フィールドで「&yen;（円）」形式の計算結果を表示したい場合に使用します。

<Heading id="autocalc_yen_function_10">YEN関数の構文</Heading>

```plaintext {copy=true}
YEN(数値, 桁数)

```

YEN関数には、引数として「数値」と「桁数」の両方を必ず指定します。

「数値」の引数に指定できるのは、数値フィールドのフィールドコード、値が数値になる計算式、または数値です。  
「桁数」の引数には、四捨五入する小数点以下の桁数を指定します。小数点の左側（整数部分）で四捨五入したい場合は、0または負の数値を指定します。

<Note>

フィールドコードは、計算式やAPIでフィールドを指定するときに使用する文字列です。フィールドごとに設定します。詳細は、次のページを参照してください。  
[フィールドコードとは](/k/ja/id/040504.html)

</Note>

<Heading id="autocalc_yen_function_20">YEN関数が使用できるフィールド</Heading>

YEN関数の結果を表示する場合は、文字列（1行）フィールドで使用します。  
「文字列（1行）フィールドの設定」ダイアログで「自動計算する」にチェックを付け、計算式を入力してください。

計算フィールドでYEN関数の結果を直接表示しようとすると、エラーになります。  
IF関数の条件式として使用するなど、関数の中であれば使用できます。

<Note>

* YEN関数の計算結果のデータ型は、文字列型になります。データ型の詳細は、次のページを参照してください。  
  [計算結果のデータ型](/k/ja/id/040500.html#autocalc_calculation_type_20)
* 計算フィールドや数値フィールドで桁区切りを表示したい場合は、フィールドの設定ダイアログで桁区切りの表示を設定します。詳細は、次のページを参照してください。
  * [桁区切りを表示する](/k/ja/id/040534.html#form_parts_number_2030)
  * [表示形式](/k/ja/id/040517.html#form_parts_calculated_2040)
* IF関数についての詳細は、次のページを参照してください。  
  [[IF関数]条件によって表示する値を変える](/k/ja/id/040505.html)

</Note>

<Heading id="autocalc_yen_function_30">計算式の例</Heading>

計算式でフィールドを指定する場合は、引数としてフィールド名ではなくフィールドコードを指定してください。  
次の例では、フィールド名とフィールドコードをすべて一致させています。

<Heading level={3} id="autocalc_yen_function_3010">小数第2位で四捨五入して「&amp;yen;（円）」形式で表示する計算式</Heading>

次の例は、「税込価格」の数値フィールドの値を小数第2位で四捨五入し、「&yen;（円）」形式で表示する計算式です。

```plaintext {copy=true}
YEN(税込価格, 1)

```

「税込価格」フィールドに「1132.85」が入力された場合、この計算式を設定したフィールドに「&yen;1,132.9」が表示されます。

<Heading level={3} id="autocalc_yen_function_3020">小数第1位で四捨五入して「&amp;yen;（円）」形式で表示する計算式</Heading>

次の例は、「税込価格」の数値フィールドの値を小数第1位で四捨五入し、「&yen;（円）」形式で表示する計算式です。

```plaintext {copy=true}
YEN(税込価格, 0)

```

「税込価格」フィールドに「1132.85」が入力された場合、この計算式を設定したフィールドに「&yen;1,133」が表示されます。

<Heading level={3} id="autocalc_yen_function_3030">一の位で四捨五入して「&amp;yen;（円）」形式で表示する計算式</Heading>

次の例は、「税込価格」の数値フィールドの値を1桁目で四捨五入し、「&yen;（円）」形式で表示する計算式です。

```plaintext {copy=true}
YEN(税込価格, -1)

```

「税込価格」フィールドに「1132.85」が入力された場合、この計算式を設定したフィールドに「&yen;1,130」が表示されます。
