レコードコメントやラベル／リッチエディターフィールドに、画像やファイルを添付する機能はありません。  

なお、同じ<Kintone />上のレコード／スレッドなどに添付した画像やファイルであれば、そのファイルを入力欄にドラッグアンドドロップして貼り付けることが可能です（コピー&ペーストでも貼り付けできます）。

<Warning>

* コピー元のファイルが添付されていた場所から削除されると、レコードコメントやラベル／リッチエディターフィールドに表示されていたファイルも表示できなくなる場合があります。  
* レコード追加／編集画面からは、画像やファイルを貼り付けできません。
* この内容は公式サポート対象外です。  
  今後のアップデートで動作しなくなる可能性があります。

</Warning>
