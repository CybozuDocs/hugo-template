メッセージの操作をしたときに出力されるログについて説明します。

## Message operation{#log_message_operation_10,class="no-trans"}

モジュールが「Message operation」である監査ログは、下記のとおりです。

### Message comment file download{#log_message_operation_1010,class="no-trans"}

アクションが「Message comment file download」である監査ログについて説明します。

#### 実行された操作{#log_message_operation_101010}

メッセージに添付されたファイルについて、次の操作をした。

* ファイルをダウンロードした
* 画像のスライドショーを表示した

#### レベル{#log_message_operation_101020}

情報

#### 「補足」に出力される内容{#log_message_operation_101030}

次の項目が出力されます。

* ログイン名
* メッセージのURL
* ファイル名

#### 出力されるログの例{#log_message_operation_101040}

```
users: [*], comment url: *, filename: *

```

ログイン名はカンマ区切りで出力されます。

## プロパティ{#log_message_operation_20}

監査ログの「補足」欄に出力されるプロパティは次のとおりです。

### comment url{#log_message_operation_2010,class="no-trans"}

メッセージのURLです。

### filename{#log_message_operation_2020,class="no-trans"}

ファイル名です。

### users{#log_message_operation_2030,class="no-trans"}

ユーザーのログイン名です。
